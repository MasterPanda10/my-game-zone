# 🎮 GAMEZONE — Игровой Портал

Современный, адаптивный одностраничный веб-сайт (лендинг), посвященный главным игровым новинкам, актуальным новостям индустрии и рейтингам игр.

🪐 **Посмотреть сайт вживую:** [https://[MasterPanda10].github.io/[my-game-zone]/](https://[MasterPanda10].github.io/[my-game-zone]/)

---

## ✨ Основные возможности сайта
* **⚡ Динамический живой поиск**: мгновенная фильтрация игр в таблице рейтингов по названию или платформе без перезагрузки страницы.
* **💬 Интерактивная лента комментариев**: пользователи могут делиться мнением. Все комментарии сохраняются в локальной памяти браузера (`LocalStorage`) и остаются на месте даже после обновления страницы.
* **❤ Система лайков**: возможность оценивать комментарии других участников сообщества в реальном времени.
* **📱 Полная адаптивность**: сайт одинаково красиво отображается на экранах мониторов, планшетов и мобильных телефонов благодаря CSS Media Queries.

---

## 🛠 Использованные технологии
* **HTML5** — структурирование контента, семантические теги.
* **CSS3** — современная темная геймерская тема, неоновые акценты, CSS Grid и Flexbox для сетки новостей.
* **Ванильный JavaScript (ES6+)** — логика работы поиска, обработка формы, лайков и интеграция с хранилищем данных.

---

## 🚀 Как запустить проект локально
1. Клонируйте репозиторий или просто скачайте файл `index.html`.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Яндекс.Браузер, Firefox, Edge).
3. Для работы сайта не требуется установка дополнительных серверов или баз данных.
